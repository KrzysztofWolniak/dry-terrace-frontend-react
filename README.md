## Run Locally

Clone the project

```bash
  git clone https://github.com/KrzysztofWolniak/dry-terrace-website-react
```

Go to the project directory

```bash
  cd dry-terrace-website-react
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
